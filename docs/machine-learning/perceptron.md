# Perceptron

## What is a Perceptron?

A perceptron is the simplest artificial neuron used for binary classification.

---

## Intuition

Imagine you want to decide whether an email is spam.

The perceptron looks at different features of the email and combines them using weights.

If the final score is greater than a threshold, it predicts **Spam**.

Otherwise it predicts **Not Spam**.
